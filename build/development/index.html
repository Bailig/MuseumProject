The project is to develop a story map strategy by using the geotagged stories from the Canada science and technology museum.

Project Background

The museum closed in 2014, and it is going to be relaunched in 2017. The museum want the it to be a fully digital interactive experience. Therefore, the old data have to be illustrated in a new digital and interactive way. For this particular project, the digital storybook with geographic names will be shown on a map with the stories.

Project Overview

Part One

The first part of the project is to geocode the geographic names, such as street name and city name, to a latitude and longitude, which is done by the student team from computer programming field.

Input
Tables have a description with geographic names in each row.
Output
Tables with two columns of data added, which are geocoded latitude and longitude.

Part Two

In the second part of the project, I created an interactive online map that locates a wide variety of technologies throughout the history of Canada. The map relies on the ESRI story map template and ArcGIS online.

Input
The tables provided by the student team with latitude and longitude.
Output
A fully customized story map template,
A series of scripts for the preparation of the data,
A practice guide
